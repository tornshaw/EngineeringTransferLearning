# kbtl_demo_multiclass_py

This document describes the Python migration of the original `kbtl_demo_multiclass.m` script.

## Source
- MATLAB: `demos/kbtl_demo_multiclass.m`
- Python: `demos/kbtl_demo_multiclass.py`

## Functionality
- Synthetic or real data ingestion
- KBTL hyperparameters set for multi-class
- Training via `models.kbtl.kbtl_train`
- Testing via `models.kbtl.kbtl_test`
- Training and testing accuracy reporting

## Usage
```bash
python demos/kbtl_demo_multiclass.py
```
